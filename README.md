### Installing with Composer ###

_Include the following in the root level of your composer.json:_

```JSON
{
    "repositories": [
        {
            "url": "https://github.com/digices-llc/mysqlr",
            "type": "vcs"
        }
    ],
    "require": {
        "digices/paqure": "0.0.3"
    },
    "autoload": {
        "psr-4": {
            "digices\\mysqlr\\": "src"
        }
    }
}

```
